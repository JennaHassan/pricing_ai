# Nafel Pricing AI Estimator

## Description
The Nafel Pricing AI Estimator is an AI-powered tool that automates construction project cost estimation for Nafel Contracting. The tool saves significant time by automating pricing calculations and becomes more accurate as it processes more project data. It is designed to streamline project budgeting and material cost tracking.

You can use the online version here:  
**https://nafelpricing.streamlit.app/**

## Features
- Process project materials from CSV files
- Generate estimated project costs
- Improves estimation accuracy over time with more data
- Integrates with tools like Excel and Google Calendar
- User-friendly and easy to adapt for new projects
- 
## How to Use the Web App
1. Visit the app:  
   **https://nafelpricing.streamlit.app/**
2. Examples to type "Estimate the price of installing 3 interpipes", check csv file for options or replace interpipes with any of these words: Statx 250g, Heat Detector, Statx 60g.
3. Click Estimate
4. The app will generate your result
